0x03_Ubuntu raid5 create and remove Linux
2023-05-19 - 2023/05/19/0x03_linux/
create raid5

Creating raid5 with sd[b-d]
sudo mdadm --create --verbose /dev/md0 --level=5 --raid-devices=3 /dev/sd[b-d]

Format the raid5 you just created
sudo mkfs.ext4 /dev/md0
if you now reboot md0 will be md127. you should follow these to avoid it
sudo mdadm --detail --scan
you get this copy it into the last line of /etc/mdadm/mdadm.conf

ARRAY /dev/md0 metadata=1.2 ....

make it effective
sudo update-initramfs -u
last step mount your md0 to /mnt/md0
sudo nano /etc/fstab

/dev/md0(Tap)/mnt/md0(Tap)ext4(Tap)defaults(Tap)0(Tap)0

reboot and run lsblk you would see the result

remove raid5

sudo nano /etc/fstab
delete the last line you have created before otherwise your system could not be started
sudo umount /mnt/raid5
and
sudo mdadm -S /dev/md0
to stop raid5
don’t forget sudo nano /etc/mdadm/mdadm.conf to delete you have created
now you can clean the md0 in every sd[b-d]
sudo mdadm --zero-superblock /dev/sd[b-d]

finally
sudo update-initramfs -u
