0x01_My-OS介绍 自制操作系统
2023-04-18 - 2023/04/18/0x01_my-os/

功能

  • 实现了运行 app，调用自己写的 API 实现输出字符串
  • 有终端输入输出界面
  • 带有计时器
  • 支持鼠标和键盘

问题

实体机鼠标有问题。

简介

这个系统是完全参考 haribote 的，算是精简版或者二次开发版的 haribote，去除了 GUI 界面，终端界面完全自己开发。这是经过一点一点的调试才得到的成果。
我想将代码搬运到 nasm + gcc 环境下运行。下面的教程可以帮助您以和我相同的效果造出这个系统。同时，我也会教您如何修改来满足您自己的需求。教程和 haribote 一样按天划分，请各位为这个项目在 GitHub 上点个 star。
