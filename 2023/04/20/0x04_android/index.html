0x04_Android实现连接MySQL Android工具类
2023-04-20 - 2023/04/20/0x04_android/

这个工具类可以实现连接mysql

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class MySQLConnectionHelper {
    private static final String DRIVER = "com.mysql.jdbc.Driver";
    private static final String HOST = "localhost:3306";
    private static final String DB_NAME = "mydatabase";
    private static final String URL = "jdbc:mysql://" + HOST + "/" + DB_NAME;
    private static final String USER = "root";
    private static final String PASSWORD = "123456";

    public static Connection getConnection() {
        Connection connection = null;
        try {
            Class.forName(DRIVER);
            connection = DriverManager.getConnection(URL, USER, PASSWORD);
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        } catch (SQLException e) {
            e.printStackTrace();
        }
        return connection;
    }
}

该类通过静态方法getConnection创建到数据库的连接并返回一个Connection对象。

您可以使用以下代码来测试该工具类：

Connection connection = MySQLConnectionHelper.getConnection();
if (connection != null) {
    System.out.println("数据库连接成功！");
} else {
    System.out.println("数据库连接失败...");
}

在这个例子中，我们简单地在控制台上打印出数据库连接结果。
要使用Java JDBC驱动连接MySQL数据库，需要在项目中添加MySQL驱动程序的依赖。
对于Android应用程序项目，你需要将以下行添加到项目的gradle文件（通常是build.gradle(Module: app)）的dependencies部分：
implementation 'mysql:mysql-connector-java:8.0.26' 这将添加MySQL`驱动程序的依赖，版本是8.0.26。如果你想使用其他版本的驱动程序，请相应地更改依赖项。

请注意，如需在Android应用程序中使用Java JDBC，您需要在应用程序中声明INTERNET权限。你可以在AndroidManifest.xml文件中添加以下行来声明此权限：

<uses-permission android:name="android.permission.INTERNET" />
